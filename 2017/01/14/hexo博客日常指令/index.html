hexo博客日常指令

发表于 2017-01-14 | | 阅读次数

新建一篇文章的两种方式

  1. 通过指令创建,在hexo目录命令行下执行 hexo new 文章名 命令,此时会自动在\Hexo\source_posts目录下生成文章名.md文件
    同时,文件内会自动生成标题,日期等内容
  2. 也可以自己写好markdown文件,然后将文件直接放在\Hexo\source_posts目录下

开启本机服务器预览博客内容

在hexo目录命令行下执行hexo server 或 hexo s

清除之前的数据可文件和public文件

在hexo目录命令行下执行hexo clean 或 hexo cl

生成静态文件到public文件夹下

在hexo目录命令行下执行hexo generate 或 hexo g

将博客部署到git服务器上

在hexo目录命令行下执行hexo deploy 或 hexo d
